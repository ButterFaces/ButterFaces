ButterFaces a smart JSF framework

Welcome to ButterFaces, a lightweight and modern JSF framework that takes the best from Twitters Bootstrap, JQuery and HTML 5 to develop fast, easy and modern web applications using Mojarra JSF 2.
Why ButterFaces?

Butter tastes good

Because everything tastes better with butter. This is where the name of ButterFaces is coming from.

HTML 5 features

ButterFaces bootstraps most of mojarras input components and allowes you to use most of html 5 features like placeholder, type, pattern and so on...

Additional features

Each input component is extended by a readonly view and has a responsive label and tooltip attributes supporting ajax and validation.

Based on Twitters Bootstrap and JQuery

Bootstrap brings up a responsive design. A lot of JQuery plugins helps to add a smart look and feel with beeing browserunabh√§ngig

Minimal invasive

ButterFaces brings it own components without changing the main artifact. Even dependencies like bootstrap and jquery are configurable.
Easy to use

One dependency

There is only one dependency you need. You can put one jar to your project.
ButterFaces is available in maven central respository so you can easily use maven to add ButterFaces to your application

No addition css and js library

ButterFaces comes up with bootstrap and jquery. You do not have to add it by yourself to your appliaction (but you could).
