b:treeBox The treeBox tag renders a HTML input element of the type "text" using Trivial Components TreeComboBox to customize rendered view. No longer use of selectitems nor converter to render a combobox - just a list of objects (and in case of a tree of nodes). Select rootNode and submit form to produce a validation error. Override toString() of Episode.java to handle rendered values or add <f:facet name="template"> {{propertyToDisplay}} </f:facet> When using facet you have to specify inputTextProperty which will be filled into the input field when the user selects an entry.
